## 又称
代理模式

## 目的
它实质是一个代表对象向外部表示某种行为，但实际上代表将该行为委托给关联对象执行。 

![alt text](./etc/delegation.png "Delegate")

## Applicability
使用委托模式来实现以下操作。

* 减少方法与类的耦合
* 这些组件的行为是相同的，但是要意识到这种情况在将来会发生变化。
